Obtaining guake-indicator

Source code for Guake indicator may be obtained from this directory

Guake-indicator Ubuntu packages may be obtained from https://launchpad.net/guake-indicator

If you want to keep-up with the latest release please consider adding my PPA. To install guake-indicator from my ppa, type in your terminal : 

sudo add-apt-repository ppa:gun101/ppa
sudo apt-get update
sudo apt-get install guake-indicator

From July 15 2014 guake-indicator is part of Debian Sid and thus of the Debian project.
Thanks to Sylvestre Ledru for sponsoring the project and to Thorsten Alteholz for reviewing it.
